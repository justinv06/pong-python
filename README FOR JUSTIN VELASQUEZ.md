## 🎯 Objetivo del Programa

El objetivo del programa es desarrollar un juego básico de Ping Pong
utilizando Python y la librería pygame. El juego permite que dos
jugadores controlen una paleta cada uno e intenten anotar puntos
evitando que la pelota salga de su lado de la pantalla.

Además, el programa demuestra el uso de estructuras de datos como: -
Tuplas - Listas - Diccionarios

------------------------------------------------------------------------

## 🕹️ Principales Funcionalidades

### 1️⃣ Inicialización del Juego

-   Se configura el tamaño de la ventana mediante una tupla.
-   Se establecen los colores del juego.
-   Se crean los objetos principales: pelota y paletas.

### 2️⃣ Movimiento de las Paletas

-   El jugador izquierdo se mueve con las teclas **W** y **S**.
-   El jugador derecho se mueve con las flechas **↑** y **↓**.

### 3️⃣ Movimiento de la Pelota

-   La pelota se mueve automáticamente.
-   Rebota cuando toca la parte superior o inferior de la pantalla.
-   Cambia de dirección cuando colisiona con una paleta.

### 4️⃣ Sistema de Puntuación

-   Se utiliza un diccionario para almacenar el puntaje de cada jugador.
-   Cuando la pelota sale por un lado de la pantalla, el jugador
    contrario obtiene un punto.
-   La pelota regresa a su posición inicial.

### 5️⃣ Uso de Estructuras de Datos

🔹 **Tuplas** - Se utilizan para valores constantes como el tamaño de
pantalla, colores y posición inicial.

🔹 **Listas** - Se utiliza una lista para almacenar las paletas y
recorrerlas con un ciclo.

🔹 **Diccionarios** - Se utiliza un diccionario para asociar cada
jugador con su puntuación.
